Sota Backup

曽田さんのBackup に対する拡張を公開します。
作者の曽田さんには連絡済みですが、まだ取り込まれていません。(2018/1/14 現在)
Backup-1.18 で 2018/3/23 に取り込まれました。

ソースコード公開場所

https://bitbucket.org/m-tsuchiyama/sota-backup で公開しています。
master ブランチは Backup-1.18a-src.zip と同じ内容になっています。

GitHub だとShiftJIS が文字化けするので Bitbucket を使っています。

拡張している機能

エラーが発生した場合と正常終了した場合のアクションを変える

feature/SeparateActionOnError ブランチ
→ Backup-1.18 で 2018/3/23 に取り込まれました。

一時的にバックアップパターンを無効にする

feature/EnableDisablePattern ブランチ
→ Backup-1.18 で 2018/3/23 に取り込まれました。

上記２つのブランチをマージしたブランチ

feature/SeparateActionOnError-EnableDisablePattern ブランチ
→ Backup-1.18 で 2018/3/23 に取り込まれました。

バイナリ公開場所

  1. Bitbucket の ブランチ の一覧から ビルド をクリックします。
  2. 一番新しいビルドを選びます。
  3. appveyor のサイトに飛ぶので ARTIFACTS をクリックします。
  4. 実行ファイルをダウンロードします。
