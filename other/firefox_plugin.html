Firefox プラグイン

概要

Firefox プラグイン 関連のリンクのリンク(覚え書き)です。

  • Firefox
    • Firefox 本体です。
    • サイト http://mozilla.jp/firefox/
  • Speed Dial
    • Opera の Speed Dial を実現するプラグインです
    • サイト https://addons.mozilla.org/ja/firefox/addon/speed-dial/
  • Tab Mix Plus
    • タブの多重化などタブ関連のカスタマイズをするプラグインです
    • サイト https://addons.mozilla.org/ja/firefox/addon/tab-mix-plus/
  • IE Tab Plus (FF 3.6+)
    • IEでしかアクセスできないサイトに IE でアクセスできるようにするプラグイン
    • サイト https://addons.mozilla.org/ja/firefox/addon/ie-tab-plus-ff-36/
