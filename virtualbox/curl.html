Install -- how to install curl

curl-7.37.1

  1. 環境変数 PATH に C:\mingw64\bin を追加する。
  2. "C:\mingw64\bin\mingw32-make.exe" mingw32 を実行する

環境変数 PATH に C:\mingw64\bin を設定しない場合
