YUV Player

  1. YUV Player をダウンロードして変換した YUV ファイルを開く。
  2. 開く際に Size メニューから Custom を選び、画面サイズを指定する。
  3. File メニューから Open を選びファイル名を指定する。
  4. File メニューから Go を選びフレーム番号を指定する。

yuv420p
