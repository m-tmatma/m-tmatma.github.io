Jenkins のビルド

Fedora 26

準備

# dnf install -y maven

ビルド

$ git clone https://github.com/jenkinsci/jenkins.git
$ cd jenkins
$ mvn clean install -pl war -am -DskipTests
参考: https://github.com/jenkinsci/jenkins/blob/master/BUILDING.TXT
./war/target/jenkins.war に生成される。

Windows

準備

maven

https://maven.apache.org/download.cgi から Binary zip archive をダウンロードする。
適当なフォルダに、解凍する

JDK

http://www.oracle.com/technetwork/java/javase/downloads/index.html から JDK をダウンロードする。
インストーラをダブルクリックしてインストールする。
環境変数 JAVA_HOME に JDK のパスを設定する。
例: C:\Program Files\Java\jdk1.8.0_144

ビルド

> git clone https://github.com/jenkinsci/jenkins.git
> cd jenkins
> "C:\apache-maven-3.5.0\bin\mvn.cmd" clean install -pl war -am -DskipTests
