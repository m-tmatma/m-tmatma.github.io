Jenkins Linux インストール (Fedora 24)

Jenkins の本家サイト から Jenkins の RPM ファイルを入手する。 
# dnf install jenkins-2.15-1.1.noarch.rpm
# chkconfig jenkins on
# /etc/init.d/jenkins start
# firewall-cmd --permanent --add-port=8080/tcp
# firewall-cmd --reload
