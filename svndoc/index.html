概要

Subversion をWindows 上 でコンパイルするのは非常に面倒なので、まず最小の構成でコンパイル する手順を説明し、徐々にオプションの機能を追加していく形で説明します。

コンパイル手順 (VC6)

  1. 最小の構成でコンパイルする (VC6)
  2. Subversion 用に Apache2を構築 (VC6)
  3. SSLのサポート付きでコンパイルする (VC6)
  4. 国際化対応付きでコンパイルする (VC6)
  5. python binding をコンパイルする (VC6)
  6. win-test.py (Subversion の回帰テストツール)

  1. 最小の構成でコンパイルする (VS2005)

  3. SSLのサポート付きでコンパイルする (VS2005)
  4. 国際化対応を入れてコンパイルする方法 (VS2005)
