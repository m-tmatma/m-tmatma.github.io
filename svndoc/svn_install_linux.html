Subversion の コンパイル (VineLinux 4.2)

概要

VineLinux 4.2 で trunk の Subversion を使う方法のメモ。

手順

root で実行 
apt-get -y install neon-devel
apt-get -y install zlib-devel
apt-get -y install sqlite3-devel
apt-get -y install apache2-devel
apt-get -y install openssl-devel
apt-get -y install swig
apt-get -y install python-devel
apt-get -y install subversion
一般ユーザーで実行 
svn co http://svn.apache.org/repos/asf/subversion/trunk
cd trunk
./autogen.sh
./configure --prefix=/usr --with-apxs=/usr/bin/apxs
make
make swig-py
root で実行 
apt-get -y remove  subversion
make install
make install-swig-py
echo /usr/lib/svn-python > /usr/lib/python2.4/site-packages/subversion.pth
