svn youngest

概要

リポジトリの HEAD リビジョン取得できるコマンドです。

これまでリポジトリがローカルディスクにあるときには svnlook youngest で取得できましたが、これを作業コピーまたは URL に対して取得できるコマンドです。

Subversion 1.9 以降で利用可能です。

ヘルプ

>svn.exe help youngest
youngest: Print the youngest revision number of a target's repository.
usage: youngest [TARGET]

  Print the revision number of the youngest revision in the repository
  with which TARGET is associated.

Valid options:
  --no-newline             : do not output trailing newline

Global options:
  --username ARG           : specify a username ARG
  --password ARG           : specify a password ARG
  --no-auth-cache          : do not cache authentication tokens
  --non-interactive        : do no interactive prompting (default is to prompt
                             only if standard input is a terminal device)
  --force-interactive      : do interactive prompting even if standard input
                             is not a terminal device
  --trust-server-cert      : accept SSL server certificates from unknown
                             certificate authorities without prompting (but only
                             with '--non-interactive')
  --config-dir ARG         : read user configuration files from directory ARG
  --config-option ARG      : set user configuration option in the format:
                                 FILE:SECTION:OPTION=[VALUE]
                             For example:
                                 servers:global:http-library=serf

使用例

作業コピーに対して実行 (引数なし) 
>svn.exe  youngest
1508252
作業コピーに対して実行 (引数あり) 
>svn.exe  youngest subversion
1508252
URL に対して実行 (Repository Root ではないパス ) 
>svn.exe  youngest http://svn.apache.org/repos/asf/subversion/trunk
1508252
URL に対して実行 (Repository Root) 
>svn.exe  youngest http://svn.apache.org/repos/asf
1508252
--no-newline オプションを指定して実行 (改行なしで出力。svnversion の -n と同じ) 
>svn.exe  youngest --no-newline http://svn.apache.org/repos/asf
1508253
