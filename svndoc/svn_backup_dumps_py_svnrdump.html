概要

このページでは、 svn-backup-dumps.py を改造して svnrdump 対応にしたカスタム版 の使用方法を説明します。

スクリプトの管理

カスタム版は GitHub で管理しています。https://github.com/m-tmatma/svn-backup-dumps

使い方

svnrdumpを使用するので URL 指定でバックアップを取ることができます。
