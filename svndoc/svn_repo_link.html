Subversion リポジトリのリンク集

概要

Subversion で管理されているプロジェクトのリンク集です。

Note: "TortoiseSVN 用リンク" に関しては以下を参照 http://tortoisesvn.net/docs/release/TortoiseSVN_ja/tsvn-repository-links.html

Subversion が依存しているコンポーネント

  • serf
    • WebDAV 用のライブラリ
    • リポジトリ http://serf.googlecode.com/svn/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://code.google.com/p/serf/
  • neon
    • WebDAV 用のライブラリ
    • リポジトリ http://svn.webdav.org/repos/projects/neon/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://www.webdav.org/neon/
  • apr
    • Apache Portable Runtime (ライブラリ集です)
    • リポジトリ http://svn.apache.org/repos/asf/apr/apr/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://apr.apache.org/
  • Apache
    • Apache Web Server
    • リポジトリ http://svn.apache.org/repos/asf/httpd/httpd/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://httpd.apache.org/
  • SWIG
    • Ruby, Python, Perl の bindings の作成用のツール
    • リポジトリ http://swig.svn.sourceforge.net/svnroot/swig/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://www.swig.org/

Subversion Client

  • Subversion
    • Subversion です。
    • リポジトリ http://svn.apache.org/repos/asf/subversion/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://subversion.apache.org
  • TortoiseSVN
    • Windowsエクスプローラ拡張を利用した Subversion GUI クライアント
    • リポジトリ http://tortoisesvn.googlecode.com/svn/trunk
    • TortoiseSVN 用リンク
    • プロジェクト (バイナリダウンロード) http://tortoisesvn.net/
    • プロジェクト (メインサイト) http://code.google.com/p/tortoisesvn/
    • デイリービルド http://nightlybuilds.tortoisesvn.net/latest/
  • rapidsvn
    • Windows, Linux, Mac 等マルチプラットフォーム対応 の Subversion GUI クライアント
    • リポジトリ http://rapidsvn.tigris.org/svn/rapidsvn/trunk (要認証)
    • TortoiseSVN 用リンク
    • プロジェクト http://rapidsvn.tigris.org/
  • SvnMergeGui
    • svnmerge.py の Windows 版GUIクライアント
    • リポジトリ http://svnmergegui.googlecode.com/svn/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://code.google.com/p/svnmergegui/

C# bindings

  • sharpsvn (Complete binding of the Subversion Client API for .Net 2.0 applications)
    • C# bindings
    • リポジトリ http://sharpsvn.googlecode.com/svn/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://code.google.com/p/sharpsvn/

Vistual Studio のSubverion プラグイン

  • ankhsvn
    • Visual Studio .NET 上で Subversion を使うためのプラグイン
    • リポジトリ http://ankhsvn.open.collab.net/svn/ankhsvn/trunk
    • リポジトリ (閲覧用) http://ankhsvn.open.collab.net/source/browse/ankhsvn/
    • TortoiseSVN 用リンク
    • プロジェクト http://ankhsvn.open.collab.net/
  • StudioTurtle - Subversion for Visual Studio 2005
    • Visual Studio 2005 上で Subversion を使うためのプラグイン
    • リポジトリ http://studioturtle.googlecode.com/svn/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://code.google.com/p/studioturtle/

プロジェクト管理ツール

  • Redmine
    • Ruby on Rails をベースに作成されたプロジェクト管理ツール
    • リポジトリ http://redmine.rubyforge.org/svn/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://www.redmine.org/
      • 参考: http://www.asahi-net.or.jp/~iu9m-tcym/redmine/
  • trac
    • Python で作成されたプロジェクト管理ツール
    • リポジトリ http://svn.edgewall.org/repos/trac/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://trac.edgewall.org/
  • svnLogBrowser
    • Web ベースのリポジトリビューアー
    • リポジトリ http://svn.svnlogbrowser.org/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://svnlogbrowser.org/
    • デモサイト http://svnlogbrowser.org/demo/

他の Version 管理ツールから Subversion への変換ツール

  • vss2svn
    • VSS -> Subversion の変換ツール
    • リポジトリ http://svn.pumacode.org/repos/vss2svn/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://www.pumacode.org/projects/vss2svn
  • cvs2svn
    • CVS -> Subversion の変換ツール
    • リポジトリ http://cvs2svn.tigris.org/svn/cvs2svn/trunk (要認証)
    • TortoiseSVN 用リンク
    • プロジェクト http://cvs2svn.tigris.org/

Subversion とは全く関係ないツール

    • Windows 上で動作するテキストエディタ
    • リポジトリ http://sakura-editor.svn.sourceforge.net/svnroot/sakura-editor/sakura/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://sakura-editor.sourceforge.net/
  • WinMerge
    • WinMerge (マージソフト)
    • リポジトリ http://winmerge.svn.sourceforge.net/svnroot/winmerge
    • TortoiseSVN 用リンク
    • プロジェクト http://winmerge.org/
      • Note: WinMerge 日本語版
  • doxygen
    • doxygen
    • リポジトリ https://doxygen.svn.sourceforge.net/svnroot/doxygen/trunk
    • TortoiseSVN 用リンク
    • プロジェクト http://www.stack.nl/~dimitri/doxygen/
  • wireshark (バケットモニタソフト)
    • wireshark (Ethereal の後継)
    • リポジトリ http://anonsvn.wireshark.org/wireshark/trunk/
    • TortoiseSVN 用リンク
    • プロジェクト http://www.wireshark.org/
    • 開発者サイト http://www.wireshark.org/develop.html
