autoprop の設定を svn_apply_autoprops.py で作業コピーに反映する

概要

Subversion では新規ファイルを登録するときに規定のプロパティを設定する機能 autoprop が あります。しかしこの機能はクライアント側の設定のため強制力はなく指定したファイルに 意図したプロパティが設定されていないことがあります。 svn_apply_autoprops.py を使うと autoprop の設定を読み込んで作業コピーに設定を 反映することができます。

セットアップ手順

入手先

svn_apply_autoprops.py は以下から入手可能です。 http://svn.apache.org/repos/asf/subversion/trunk/contrib/client-side/svn_apply_autoprops.py

Subversion のソースコード配布ファイルの中にも含まれています。

Subvesion Python bindings のインストール

svn_apply_autoprops.py は Subversion Python bindings を使っているので Python bindings をインストールする必要があります。

参照

  • subversion/bindings/swig/INSTALL
  • Python binding をコンパイルする方法 (VC6)
