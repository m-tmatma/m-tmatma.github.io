Visual Studio プラグイン ForceUTF8/ForceUTF8 with BOM

Visual Studio は プラグインを使って拡張することができます。
ForceUTF8/ForceUTF8 with BOM を紹介します。
vsix ファイルをダブルクリックしてインストールします。インストールするときには Visual Studio は終了しておきます。

ForceUTF8

保存するときに UTF8 なしで保存するプラグインです。

Visual Studio でファイルを保存するときに UTF8 で保存するのを強制するプラグインです。

以下からダウンロードできます。
ForceUTF8 (with BOM)

ソースコードは GitHub で公開されています。
https://github.com/vilic/vs-force-utf8 対応 Visual Studio は 2013/2015 です。

ForceUTF8 with BOM

日本語での解説サイト
Visual Studio 2013/2015 で Shift JIS を UTF-8 に自動変換してコードを保存する

for 2013/2015

ForceUTF8 を改造して保存するときに UTF8 BOM 付きで保存するように改造したプラグイン for 2013/2015 です。

以下からダウンロードできます。
ForceUTF8 (with BOM)

ソースコードは GitHub で公開されています。
https://github.com/jz5/vs-force-utf8
対応 Visual Studio は 2013/2015 です。
