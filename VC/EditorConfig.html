Visual Studio プラグイン EditorConfig/EditorConfig Language Service

EditorConfig とは?

EditorConfig とは .editorconfig というファイルで各種エディタや IDE で コーディングスタイルを定義して、一貫したコーディングスタイルを維持するための仕組みです。
EditorConfig プロジェクトは .editorconfig のフォーマット定義と実現するためのプラグインを提供しています。

EditorConfig をサポートしているエディタや IDE のリストです。

Visual Studio 2010 ～ Visual Studio 2015 用のプラグイン

EditorConfig は .editorconfig をロードして定義したスタイルのとおりに動作する Visual Studio 2010 ～ Visual Studio 2015 用のプラグインです。

ソースコードは https://github.com/editorconfig/editorconfig-visualstudio で 公開されています。

.editorconfig の編集を支援するプラグインは提供されていないようです。

Visual Studio 2017 用のプラグイン

Visual Studio 2017 はネイティブで .editorconfig をサポートしていますが、編集機能は提供されていません。

EditorConfig Language Service (.editorconfig の編集を支援するプラグイン)

EditorConfig Language Service は .editorconfig の編集を支援するための Visual Studio 2017 用のプラグインです。

ソースコードは https://github.com/madskristensen/EditorConfigLanguage で 公開されています。

.editorconfig の書き方

  • どんなエディタでもEditorConfigを使ってコードの統一性を高める
  • 行末の空白は EditorConfig で始末しましょう
  • チーム開発に効く環境構築術 第1回 EditorConfigのススメ
  • Projects Using EditorConfig

Create a plugin でいろんな開発言語のパーサーのライブラリが提供されています。
