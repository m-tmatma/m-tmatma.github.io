CopyWithLineNumbers (行番号付きでコピーするプラグイン) のマニュアル

  • English
  • 日本語

GitHub

https://github.com/m-tmatma/CopyWithLineNumbers

インストール方法

  1. ここからダウンロードできます
  2. すべての Visual Studio を閉じます
  3. CopyWithLineNumbers.vsix をダブルクリックします

使い方

  1. アクティブドキュメントで選択します
  2. アクティブドキュメントで右クリックします
  3. "Copy With Line Numbers" を選択します

カスタマイズ方法

テンプレートによりコピーするテキストをカスタマイズできます

テンプレート用変数

変数名 説明 利用可能バージョン
{FileName} ディレクトリ名を含まないファイル名 1.2 以降
{FullPath} 絶対パス 1.2 以降
{RelativePath} ソリューションディレクトリからの相対パス 1.2 以降
{TopLineNumber} 選択部分の最初の行番号 1.2 以降
{BottomLineNumber} 選択部分の最後の行番号 1.2 以降
{Selection} 選択部分の各行に行番号を付与したテキスト 1.2 以降
{SelectionNoTab} 選択部分の各行に行番号を付与したテキスト。タブをスペースに置換したテキスト 1.3 以降

テンプレートの設定方法

  1. ツールメニューを選択します
  2. オプション(O)... を選択します
  3. CopyWithLineNumbers を選択します
  4. コンボボックスから挿入したい項目を選択します
  5. Insert ボタンを押します

改行を入力したい場合 "Insert EOL" ボタンを押します
初期設定に戻したい場合、"Set Default" ボタンを押します。
