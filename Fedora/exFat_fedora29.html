exFat を使う (Fedora 29)

Fedora 29 で exFat はデフォルトで有効になっていないため、認識できるようにする方法を説明します。

インストール方法

wget http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-29.noarch.rpm
sudo rpm -Uvh rpmfusion-free-release-29.noarch.rpm
sudo dnf install fuse-exfat  -y
sudo dnf install exfat-utils -y

説明

  • 何らかの問題で Fedora に含めることができないソフトウェアのために RPM Fusion というリポジトリがあります。
  • http://download1.rpmfusion.org/free/fedora/ から Fedora の対応するバージョンのファイルを ダウンロードしてインストールことで、RPM Fusion のソフトウェアをインストールできます。
  • fuse-exfat というパッケージをインストール することで exFat を認識します。
  • exfat-utils というパッケージをインストール することで exFat 関連のツールが利用可能になります。
