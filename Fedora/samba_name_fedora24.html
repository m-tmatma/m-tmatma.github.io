samba 名でアクセスする (Fedora 24)

samba のインストール 
[root@localhost ~]# dnf install -y samba.x86_64
smb, nmb サービスの有効化 
[root@localhost ~]# systemctl enable smb.service
[root@localhost ~]# systemctl enable nmb.service
smb, nmb サービスの開始 
[root@localhost ~]# systemctl start smb.service
[root@localhost ~]# systemctl start nmb.service
バックアップを作成 
[root@localhost ~]# cp /etc/samba/smb.conf /etc/samba/smb.conf.org
smb.conf の編集 
[root@localhost ~]# vi /etc/samba/smb.conf
マシン名 linux でアクセスしたい場合 /etc/samba/smb.conf 
[global]
netbios name = linux
.........

[root@localhost ~]# systemctl restart nmb.service
[root@localhost ~]# systemctl restart smb.service
/etc/hostname の編集 (オプション) 
[root@localhost ~]# vi /etc/hostname
[root@localhost ~]# cat /etc/hostname
linux
これで ttssh や winscp などで linux という名前でアクセスできるようになる。
