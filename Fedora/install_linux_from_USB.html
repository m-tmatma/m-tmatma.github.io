USB メモリに Linux 用の iso イメージを書き込む

  1. USB メモリを挿す
  2. rufusをダウンロードする
  3. rufus-x.xx.exe を起動する
  4. FreeDOS のところを ISO イメージに変更する
  5. 右のアイコンをクリックする
  6. ISO イメージを指定する
  7. スタートを押す
  8. ISO イメージモードを選択した状態で OK を押す
