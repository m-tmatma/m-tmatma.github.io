USB メモリから Linux をインストール

  1. USB メモリを挿す
  2. USB Image Tool をダウンロードする
  3. USB Image Tool.exe を起動する
  4. ドライブを選択する
  5. Restore ボタンを押す
  6. Linux の ISO イメージを指定する (私が確認したのは Fedora 20)
  7. USB メモリからブートできるように BIOS を設定する
  8. USB メモリからブートするので Linux のインストールを行う

USB メモリのリセット

インストールに使用した USB メモリを他の用途に使用する場合リセットを行う
  1. USB メモリを挿す
  2. USB Image Tool.exe を起動する
  3. ドライブを選択する
  4. Reset ボタンを押す
  5. エクスプローラでドライブを選択するとフォーマットを促されるのでフォーマットする
