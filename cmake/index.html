cmake

cmake とは?

cmake とはオープンソースのクロスプラットフォームのビルドツールです。
ソースコードは GitHub で公開されています。
  • ビルドツールとは言っても別のビルドツールのプロジェクトファイルなどを生成するジェネレーターです。
  • Windows, Mac, Linux などで動作します。
  • Visual Studio, Makefile, Xcode, Ninja などに対応しています。
  • cmake ではソースコードディレクトリとビルドディレクトリを別々にすることができます。
  • CMakeLists.txt というファイルでビルドのための設定を行います。(最後に s がついていることに注意が必要)
  • バイナリは https://cmake.org/download/ からダウンロードできます
  • ver 3.8 から C#もサポートされています。(未検証)
